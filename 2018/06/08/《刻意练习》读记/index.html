《刻意练习》读记

  • 读书
  • 2018-06-08

概述

  • 用PDCA将化学反应式的“不可能”过程，精心分解细化为一系列目标和步骤，用于刻意练习，暗合波浪式前进的哲学规律；
  • QC7工具像组合拳，可帮助达成目标；

可行性

大脑有巨大的优化潜力，心理表征可通过刻意练习创建、完善、改进，帮助在生活和工作上练就杰出技能，成为杰出人物；

刻意练习有下列特征：

  1. 明确专注，可测量的目标；
  2. 完善的训练方法和步骤，明晰的测控标准；

    大脑和身体一样，对于处在舒适区之外却离得并不太远的“甜蜜点”上的挑战，改变最为迅速。

  3. 长期有意刻苦的练习，小步、专注、快速反馈纠正；
  4. 创建改进心理表征，用以解释、组织、比较信息和计划决策，以纠正错误、达成练习目标；
  5. 最好知道谁是优秀的导师，找到优秀的导师；
    1. 优秀的导师
      • 必须已经达到一定的水平；
      • 有一些可以传授给别人的有益的练习方法。
    2. 自己设计练习方法
      1. 更短的练习，更好的注意力
      2. 将技能分解
        将技能分解成一些组成部分，以便反复地练习，并且有效地分析、确定你的不足之处，然后想出各种办法来解决它们。
      3. 跨越停滞阶段
        • 新的方式挑战自己；
        • 攻克特定的弱点；
      4. 保持动机
        • 强化继续前行的理由；
        • 弱化停下脚步的理由；
      5. 精心设置目标
        将漫长的旅程分解成一系列可控的目标，并且每次只关注它们中的一个。
        甚至可以在每次达到一个目标时，给自己小小的奖励。
