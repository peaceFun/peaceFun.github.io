搭建Hexo，在不同机器上编辑博客

  • 搭建hexo
  • 2017-06-03

在有git环境，可连github的网络机器上，无缝无痛更新hexo博客。具体包括：

  1. 可取回hexo source，并剔除node_modules目录；
  2. 可取回theme目录，并剔除对应的node_module目录；
  3. 另外机器上的编辑，可正常upload；

安装NodeJS

下载NodeJS安装包
设置taobao registry

  1. npm install -g nrm –registry=https://registry.npm.taobao.org
  2. nrm use taobao

安装hexo

  1. npm i hexo -g
  2. hexo init yourblogpath
  3. cd yourblogpath
  4. hexo clean
  5. hexo g
  6. hexo s
  7. hexo d

Git环境

安装GitforWindows

配置github.com的hosts

git常用操作

  1. git init
  2. git status
  3. git add –all
  4. git commit -m ‘test’
  5. git remote add origin git@github.com:peaceFun/hexo-src.git
  6. git push -f –set-upstream origin master

测试连接github.com

打开Git Bash环境

  1. ssh -T git@github.com

permission denied问题排除

git下载博客内容

  1. git clone -b hexo-src https://github.com/peaceFun/hexo-src.git hexoblog
  2. cd hexoblog
  3. npm i
  4. 内容文件编辑
  5. hexo clean
  6. hexo g
  7. hexo s
  8. 检查页面显示正常
  9. hexo clean
  10. git add –all
  11. git commit -m ‘something…’
  12. git push -f –set-upstream origin hexo-src:hexo-src
  13. hexo d

打开查看更新后的Geo-I站点
